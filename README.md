# xgh marketplace

Claude Code plugin marketplace for **xgh** — the developer's cockpit.

## Install

```bash
claude plugin marketplace add ipedro/xgh-marketplace
claude plugin install xgh
claude plugin install lossless-claude
```

Then run first-time setup:

```
/xgh-init
```

## Plugins

| Plugin | Description |
|--------|-------------|
| **xgh** | Memory, compression, context efficiency, and dev methodology — 25 commands, 22 skills |
| **lossless-claude** | DAG-based summarization that preserves every message across sessions |
